# Código Fuente (`src/`)

Esta carpeta contiene todo el código fuente del juego, organizado por capas.

---

## ¿Qué hace cada carpeta?

| Carpeta | Descripción |
|----------|-------------|
| **modelo/** | Define los datos del juego: civilizaciones, jugador, unidades, misiones y conexiones. |
| **vista/** | Maneja la interacción con el usuario a través de la consola. |
| **controlador/** | Orquesta el flujo del juego: turnos, acciones y condiciones de victoria/derrota. |
| **combate/** | Contiene la lógica de combate por turnos y el cálculo de estadísticas del jugador. |
| **grafo/** | Implementa el grafo dirigido que conecta las 36 civilizaciones y verifica su conexidad. |
